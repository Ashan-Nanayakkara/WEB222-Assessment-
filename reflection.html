Explain the concept of Responsive Design

So when it comes to theResponsive design is a method of creating web pages that use flexible layouts, flexible graphics, and media queries in the form of cascading style sheets. The purpose of responsive design is to create web pages that recognize the size and orientation of the visitor's screen and adjust the layout appropriately.
Responsive design seems to be a rather retro approach to website design and development that addresses many of the design issues that have arisen as a result of the growth of new sorts of mobile devices. Instead of fixed-width settings, responsive design websites employ X and Y coordinates on a grid for design and mathematical percentages for pictures. Using percentages so rather than fixed-width parameters with a grid layout results in a somewhat more fluid layout that adapts to the size of the display.
The developer may determine when a certain style takes effect using media queries, a component of cascading style sheets (CSS). A media query in CSS2, for example, will supply printer-friendly style sheets if they are requested. CSS3 offers improved query capabilities, allowing style sheets to be tailored to the display of a device and served as a desktop, tablet, or smartphone style sheet based on the query result.


How does responsive design relate to web technologies

Responsive design is a method of creating web properties such that the device on which they are viewed dictates how the site is presented. It's usually done according to the "mobile first" philosophy, which means that the experience is first designed on mobile platforms like smartphones and tablets, then scaled up to bigger displays.
The necessity for this strategy stems from the realization that an increasing number of people are browsing the mobile web, and that smartphone internet access currently outnumbers conventional PC access in several areas. Touchscreens, GPS data, accelerometers, and other features on mobile devices are also available in desktop/laptop settings.
